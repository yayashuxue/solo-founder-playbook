# Solo Founder Playbook

**6 startup skills for solo founders — powered by patterns mined from 101 [Starter Story](https://www.youtube.com/@starterstory) founder interviews.**

A data-backed startup advisor in your terminal (or in your browser). Every recommendation comes from real patterns extracted from 101 long-form founder interviews — not AI vibes.

→ **Try it in 30 seconds, no install:** [solo-founder-playbook.vercel.app/try](https://solo-founder-playbook.vercel.app/try)

## Install

### Claude Code (recommended — plugin marketplace)

```
/plugin marketplace add yayashuxue/solo-founder-playbook
/plugin install solo-founder-playbook
```

All 6 skills become available immediately. No restart needed.

### OpenAI Codex (or fallback for Claude Code)

```bash
git clone --single-branch --depth 1 https://github.com/yayashuxue/solo-founder-playbook.git ~/.codex/skills/solo-founder-playbook && bash ~/.codex/skills/solo-founder-playbook/install-codex.sh
```

Restart Codex to pick up new skills.

### Update / Uninstall

**Update (plugin):** `/plugin update solo-founder-playbook` (Claude Code)
**Update (clone):** `cd ~/.codex/skills/solo-founder-playbook && git pull && bash install-codex.sh`
**Uninstall (plugin):** `/plugin uninstall solo-founder-playbook`
**Uninstall (clone):** `rm -rf ~/.codex/skills/solo-{analyze,failures,growth,patterns,playbook,roast} ~/.codex/skills/knowledge ~/.codex/skills/solo-founder-playbook`

## What's in the Data

All data was generated by the **[Starter Story Pipeline](https://github.com/yayashuxue/starterstory-pipeline)** — a 5-step process that:

1. Pulls all video URLs from a YouTube channel
2. Downloads transcripts
3. Categorizes each video into 11 startup topics
4. Summarizes and refines the raw analysis
5. Extracts cross-video patterns and keywords

We ran it on 101 [Starter Story](https://www.youtube.com/@starterstory) videos. The result:

- **11 categories**: Idea Generation, Validation, Building, Launch, Marketing, Growth, Monetization, Challenges/Failures, Team, Costs, Advice
- **80+ patterns** with frequency data
- **300+ keywords/tools** mentioned by founders

**Key findings:**
- 49.5% of founders openly discuss failure (long-form ≠ Twitter narrative)
- 74.3% use AI as a core tool
- Organic channels outperform paid 3.5:1
- Top idea origin: solving a personal problem (32/101)
- Top failure modes: multiple failures before success (15), financial crisis (13), burnout & isolation (12), building without validation (11)

Want to run the pipeline yourself or adapt it for a different channel? Check out the **[pipeline repo](https://github.com/yayashuxue/starterstory-pipeline)**.
